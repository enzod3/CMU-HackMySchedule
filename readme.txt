Project Name: 
HackMySchedule

Description:
My project is a schedule planner to assist in the process of planning a 
schedule at Carnegie Mellon. It will help you tune your schedule to the best ability
by allowing you to rank all possible course combinations and see which works best
for you.

How to run:
from the main directory
cd into the src folder
and run "python main.py"

Libraries to install:
cmu_graphics
pandas

Shortcuts:
you can add courses and course groups to the
./data/courses.txt file to have courses
add on open
